# 🐍 Snake Game Moderno

Um jogo da cobrinha desenvolvido em **HTML, CSS e JavaScript puro**, executado diretamente no navegador sem dependências externas.

## 🎮 Sobre o Projeto

Este projeto recria o clássico jogo da cobrinha com uma interface moderna inspirada em dashboards e aplicações contemporâneas.

O jogador deve controlar a cobra, coletar alimentos e evitar colisões com as paredes ou com o próprio corpo.

---

## ✨ Funcionalidades

- 🐍 Jogabilidade clássica da cobrinha
- 🎯 Sistema de pontuação em tempo real
- 🏆 Sistema de recorde (High Score)
- 💾 Salvamento automático do recorde usando Local Storage
- 🍎 Geração aleatória de alimentos
- 🎨 Interface moderna com visual dark mode
- ✨ Efeitos visuais e sombras suaves
- 📱 Estrutura leve e responsiva
- 🔄 Reinício rápido da partida
- ⌨️ Controle por teclado (setas direcionais)

---

## 📸 Tela do Jogo

### Menu Inicial

## 🧠 Regras do Jogo

1. Controle a cobra usando as setas do teclado.
2. Colete os alimentos vermelhos.
3. Cada alimento aumenta a pontuação.
4. A cobra cresce conforme se alimenta.
5. O jogo termina quando:
   - 🚧 A cobra colide com a parede;
   - 🐍 A cobra colide com o próprio corpo.
